# Motor de Recomendações com Neo4j

## Descrição
Projeto desenvolvido para a disciplina de Sistemas de Gestão de Bases de Dados II.

## Tecnologias
- Python
- Neo4j
- Faker
- Neo4j Driver

## Instalação
pip install neo4j
pip install faker

## Execução

### Gerar dados
python gerar_dados.py

### Executar recomendações
python recomendacao.py

- gerar_dados.py
- recomendacao.py
